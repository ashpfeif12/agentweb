The spec

agent.json

One file at your domain root. Tells agents what you do, what actions are available, how to authenticate, and what rules to follow.

Fields
{
  "name": "Acme Fashion",
  "spec_version": "1.0",
  "description": "Premium retailer",
  "capabilities": [
    {
      "name": "search_products",
      "type": "query"
    },
    {
      "name": "place_order",
      "type": "action"
    }
  ],
  "brand_voice": {
    "tone": "warm, never pushy"
  },
  "endpoints": {
    "mcp": "mcp.acme.com/sse"
  }
}
Quick start

Generate your agent.json

Pick your industry. Get capabilities, brand voice, and policies pre-filled.

$ npx agentweb init --industry retail --output agent.json
> Created agent.json with 5 capabilities for retail
